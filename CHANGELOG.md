# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-03

### Added

- Initial release of the Traveldotcom system.
- Role-based authentication (Admin, Advanced, User).
- Vacation management CRUD.
- Booking system with status workflow.
- Review system with admin approval.
- GitHub Actions CI workflow.
- Documentation (README, CONTRIBUTING, SECURITY).

### Changed

- Refactored entire codebase to English naming conventions.
- Standardized code comments to Javadoc format.
